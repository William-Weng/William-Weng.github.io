【Flutter 1.5】天啊，Flutter居然可以寫WebApp了耶

  • dart
  • flutter
  • install
  • webapp

前言

恐怖喔，恐怖到了極點喔，沒有想到短短數日，除了Flutter已經變成1.5版之外，而且Dart也變成了3.0版，更恐怖是它開始可以寫WebApp了耶，雖然Dart一開始就是想要取代JavaScript的，但是因為TypeScript可以原生支援JavaScript，所以就敗下陣來，而後在Flutter的加持下又重新受到大家的關注，如今Flutter藉由Dart可轉譯為JavaScript的特性，開始計畫性的侵蝕WebApp。好吧，就讓我們來做個簡單的WebApp吧。

安裝過程

更新FlutterSDK

  • 請先將FlutterSDK更新到1.5版以上

    安裝webdev

  • 這個很重要，沒有安裝這個package的話就不能用了

開啟VSCode

  • 利用VSCode開啟指令輸入框(⌘⇧P)後，輸入Flutter: New Web Project後按Enter

    專案命名

  • 接下來就是儲存專案的名稱，只能是英文小寫(網頁好像都是小寫字？)

    儲存專案

    初始化

  • 新增專案完成之後就會開始初始化

    啟動WebApp

  • 開啟Terminal(⇧ⓢ)後，輸入以下指令就可以啟動WebApp了

測試WebApp

  • 在網頁上的localhost:8080就可以看到WebApp的長相了

後記

  • 什麼？這樣就介紹完了嗎？俗話說的好：『師父領進門，修行在個人』，人生還是要靠自己去體會的，是吧？最重要的是，我也不會呀…XD。
